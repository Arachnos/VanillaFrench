VanillaFrench

French Translation of Vanilla Forums / Traduction française du système de gestion de forums Vanilla

Download this project as a .zip file Download this project as a tar.gz file

Conçu pour être flexible et d'intégration facile, Vanilla est la meilleure et la plus puissante solution de gestion de communauté au monde ! C'est donc tout naturellement que j'ai décidé de me lancer dans la traduction française de ce système de gestion de forum.

Comment installer la traduction ?

Pour installer la traduction, téléchargez la dernière version de la traduction à l'adresse suivante : http://vanillaforums.org/addon/french-locale

Dézippez l'archive et placez le dossier /French/ dans le dossier /locales/ à la racine de votre installation de Vanilla

Comment participer au projet ?

  1. Inscrivez-vous dans la communauté Vanilla, si ce n'est déjà fait, à l'adresse suivante : http://vanillaforums.org/discussions
  2. Écrivez-moi un petit message (@Arachnos)

Seulement si vous savez utiliser GitHub !

  1. Cliquez sur le bouton "Fork on Github" en haut de cette page
  2. Envoyez vos modifications via des commits dans votre projet et cliquez ensuite sur "Pull request" dans votre projet
